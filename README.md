MyStudybuddy : Personal Study Dashboard.

✔ Shows student info
✔ Tracks assignments, presentations, exams
✔ Organizes subjects & courses
✔ Calculates GPA and CGPA
✔ Helps students plan and improve their study routine

Instruction to Test Login
1. Register first before sign up:
-Name, Email, Password
-The account is saved in localStorage
2. Login
-Enter the same email and password you used during registration
-if correct, redirect to Home Page
-if incorrect, "invalid email or password"
